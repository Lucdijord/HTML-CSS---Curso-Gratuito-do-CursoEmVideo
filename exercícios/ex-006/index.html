Principais formatações

negrito / destaque

Nesta frase temos um termo em negrito usando a tag <b> .

Já nesta frase temos um termo em destaque usando o termo <strong>.

Itálico / ênfase

nesta frase temos um termo em ítalico usando a tag <i> (não semantica)

nesta frase, temos um termo em ênfase usando a tag <em>

texto marcado

podemos também criar um texto marcado usando a tag <mark>

podemos botar o código ''mark {}'' no head dentro de uma tag <style>, assim fazendo uma alteração geral toda vez que utilizarmos o <mark>

texto grande e pequeno

estamos criando um texto pequeno utilizando a tag <small>e um texto grande utilizando a tag <big>, porém é recomendado o uso de CSS, já que a tag BIG foi descontinuada.

texto deletado

podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. Utilizamos a tag <del>.

texto inserido

podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Utilizamos a tag <ins>.

texto sobrescrito

para inserir coisas do tipo x20+3. Utilizamos a tag <sup> no 20.

texto subscrito

para inserir coisas do tipo H20. Utilizamos a tag <sub> no 2.
