Código-fonte

O comando document.getElementById('teste') é escrito em linguagem JavaScript. A tag <code> serve para deixar a palavra monoespaçada. Estava desta forma antes document.getElementById('teste'). Ela é ótima para códigos, porém não faz identação, olhe o exemplo sem identação.

num = int(input('digite um número')) if num % 2 == 0: print(f'O número {num} é PAR') else: print(f'O número {num} é ÍMPAR') print('Fím do programa')

Agora, olhe o exemplo usando <code>, com o auxílio da tag <pre>, que ira fazer a identação do código. 

        
    num = int(input('digite um número'))
if num % 2 == 0:
    print(f'O número {num} é PAR')
else:
    print(f'O número {num} é ÍMPAR')
print('Fím do programa')
